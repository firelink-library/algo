## 2. Desafios

1. Implemente a representação por **matriz de adjacências** utilizando a
   linguagem C++;
2. Implemente a representação por **lista de adjacências** utilizando a
   linguagem C++;
3. Implemente uma versão da **lista de adjacências** que, em vez de uma lista
   ligada de *arestas*, usa um **hash map** para cada *vértice*.
